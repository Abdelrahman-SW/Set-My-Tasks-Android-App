<h2 align="left">About :</h2>

Set my tasks App is a very useful and powerful app to add tasks and manage your time you can add a one time task or a repeating task on the other side you can add tasks with location 📍📍

mark your Location enter details of your Task and choose when you want to be notified of this task when your enter ⏎ this location? or exit 🔚 ? or stay for a specified time you choose. ,, not only that see more dowm .. ⬇️

<h2 align="left">Tools & Technloges Used in The App : </h2>

-- Alarm Manager SDK : For Manage Tasks By Time And Data And Fire Alarms

-- Google Geofencing API : To Add Tasks By Location And Set A Circle Area Around The Location With Raduis

-- Google Maps SDK : For Select Task Location

-- Other Tools : Room Database , Work Manager , Notification Manager , Broadcast Receivers ..
